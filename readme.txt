== Description ==

&#8220;<strong>HTML5 Cumulus</strong>&#8221; is a modern (HTML5 canvas tag based) version of the classic old &#8220;<strong>WP-Cumulus</strong>&#8221; plugin.

The plugin's widget gives you a nice-looking 3D tag cloud in the sidebar that will work in all modern browsers, including mobile ones.


[DEMO](https://html5-cumulus.ru)


If you liked my plugin, please <strong>rate</strong> it.

The plugin uses [TagCanvas](http://www.goat1000.com/tagcanvas.php) jQuery plugin by Graham Breach.


== Frequently Asked Questions ==

= Installation =

1. Visit &#8220;<strong>Plugins\Add New</strong>&#8221;.
2. Search for &#8220;<strong>HTML5 Cumulus</strong>&#8221;.
3. Install and activate the plugin.
4. Add the plugin's widget on the &#8220;<strong>Appearance\Widgets</strong>&#8221; page.
5. That's all.

Additional plugin settings are available in &#8220;<strong>Settings\HTML5 Cumulus</strong>&#8221;.

= Does the plugin support localization? =

Yes, please use [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/html5-cumulus).
